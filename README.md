# openGL 学习笔记
## macos搭建开发环境
- 系统版本  `10.15.7`
- 编辑器 `vscode`
  ### 下载安装库
  使用brew 下载 glew 和glfw，`brew install glew`和`brew install glfw3`
  ### 配置GLAD
  GLAD  是一个开源的库，使用[在线服务](https://glad.dav1d.de/)服务配置，Language选择C/C++，Profile 选择Core,然后Generate一个zip文件，将zip文件下载下来解压到本地工程目录使用。
  ### 测试环境
  使用demo1中的文件测试本地环境
